## Segment Time vs Reference Lap

This section lists the fastest and slowest segments compared to the reference lap.
Negative time values indicate a gain versus the reference lap.
Positive time values indicate a loss versus the reference lap.

## Low-Confidence Timing Differences

RaceCoach suppresses low-confidence timing differences from:
- Run Summary
- Next Run Focus
- Top Opportunities

A segment may be labeled:

    timing loss unexplained

when segment time is slower but the telemetry does not identify a clear driving deficit.

These segments remain visible in the Segment Table for awareness but are not used for coaching recommendations.
